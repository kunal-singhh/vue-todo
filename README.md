# Vue 3 TodoList App with TailwindCSS

Vue , Vitejs , TailwindCSS, JSON-server
Hosted on Vercel and Backend on Glitch

## Setup Project Locally

```js
    git clone 'https://github.com/singhkunal2050/vue-todo.git'
    npm install
    npm run backend
    npm run dev

```
